    Design: communicate in new ways

    The last decades, our way of communcation changed drastically. From letters to emails, from morse code to text, and even more extreme examples can be thought of. With this change, our mind has also changed. Where our ancestors painted the walls of there caves, we now paint our screens with illustrations and other digital media (not literally ofcourse). The study Geo Media & Design prepares us for this new way of communicating by teaching us design principles which we apply in the design of infographics, posters, animations and user-friendly interfaces. Webdesign and cartography are no strangers to us either. And many more things are possible...

    User Interface Project Dashboard

    The rise of Big Data is ever going on, which is not necessarily a bad thing. The only downside of this much data is that it gets crowded and hard to understand. That's where user interface design comes in. By following a set of mantras (not rules, they are free to be interpreted any way the designer likes) a user-friendly interface can be designed. We did this for a project concerning a dahsboard. By stepping into the users shoes and listening to their wants and needs, a user-friendly interface is born! To the left is a workflow we made using wireframes. These are the bare bones of the dashboard, the markup is yet to be made.

    Organizational Theme ArcGIS Online

    As stated before, data is playing a bigger part in our lives everyday. Consequently, the amount of applications used to display this data is also growing with a rapid speed. The need for templates within a company is great and we came up with a temporary solution. As many applications to display data are made with the help of ArcGIS Online, we thought it would be best that these applications looked the same. Meaning, the same color scheme, logo, font etc. This is possible in ArcGIS Online by creating a custom font and setting this as the organizational theme, so everyone can use it! This theme is made based on the visual guidelines of Sweco Norge AS. The next step? A fully customized template using the Sweco font and other tricks that promote branding in all the best ways!

    ESRI Field Operations

    At a company like Sweco Norge AS, collecting data is an important part of the work. It is important that this data is collected in a structurized way, so it is easier to work with. We were asked to take a look at two applications provided by ESRI: the Collector App and Survey 123. During this quest, we came across something that makes these apps even more powerful: Field Operations by ESRI. This is an entire workflow consisting of ESRI ArcGIS Apps that work together to make collecting data a lot easier! To present this, we made a Power Point Presentation as well as several infographics.
